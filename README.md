# Slot Machine

Technologies:

- TypeScript
- Vue 3
- Vuetify
- Pinia
- Vitest
- Vue Testing Library

## Project setup

Requires:

- Node 16.20 or above
- PNPM 8 or above

```
# pnpm
pnpm install
```

### Compiles and hot-reloads for development

```
# pnpm
pnpm dev
```

### Compiles and minifies for production

```
# pnpm
pnpm build
```
